# Buffoonery
A Balatro mod that aims to add vanilla-adjacent content while maintaining its own style:

- 34 Jokers
  > 9 Common<br> 10 Uncommon<br> 5 Rare<br> 9 Special*<br> 1 Legendary
- 4 Decks
  > With 4 respective [Card Sleeves](https://github.com/larswijn/CardSleeves)
- 2 Enhancements
  > Applied by 1 Tarot Card
- 2 Stakes
  > After Gold Stake

<br>**<ins>Recommended</ins>: Download the [latest Release](https://github.com/pinkmaggit-hub/Buffoonery/releases)!**

# Content
For more Information, keep on scrolling or check out the [Buffoonery Wiki](https://balatromods.miraheze.org/wiki/Buffoonery)!

### Sleeves
![Buffoonery Sleeves](repo/Sleeves.png)

[Card Sleeves](https://github.com/larswijn/CardSleeves) apply their respective Decks effect.
Combined with their Deck, they apply a different Effect:

| Name | Deck + Sleeve |
| ---- | ---- |
| JimboStation Sleeve | Start run with<br>**Deluxe Memory Card**<br>Joker instead |
| Galloping Sleeve | Start run with<br>**White Pony**<br>Joker instead |
| Porcelain Sleeve | Start run with<br>**Dorkshire Gold**<br>Joker instead |
| Sandstone Sleeve | 37.5% lowered Win Ante<br>X3 base Blind size |

## Enhancements
![Buffoonery Enhancements](repo/Enhancements.png)

| Name | Effect |
| ---- | ---- |
| Tarot:<br>Nobility | Enhances 1 selected card<br>into a Porcelain Card<br>or Royal Porcelain Card<br>if you own at least $50 |
| Porcelain Card | Fragile*<br>X1.75 Chips |
| Royal Porcelain Card | Fragile*<br>X2.0 Chips and<br>considered<br>as Face-Card |

***Fragile**:
Porcelain Cards are destroyed when
played hand contains more than 3 cards.
